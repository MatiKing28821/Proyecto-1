# STRK — Tienda Deportiva Urbana

Sitio web estático de tienda de ropa deportiva streetwear, desarrollado como parte de la **Evaluación Sumativa 1**. Aplica HTML5 semántico, CSS3 con Flexbox, imágenes reales (Unsplash), formulario de contacto y control de versiones con Git.

## 📋 Prompts utilizados

### Prompt 1 — Estructura HTML de tienda streetwear
**¿Para qué?** Generar el HTML5 semántico completo con todas las secciones.

> "Crea el HTML5 de una tienda de ropa deportiva urbana llamada STRK. Debe tener: header con logo y nav, sección hero con imagen de fondo real de Unsplash de un atleta urbano, sección novedades con grid asimétrico, sección de 6 productos con imágenes de Unsplash, galería lookbook, banner promocional, formulario de contacto y footer. Todas las imágenes deben ser de Unsplash con URLs reales."

---

### Prompt 2 — CSS urbano y colorido
**¿Para qué?** Diseñar la identidad visual streetwear con CSS3 y Flexbox.

> "Diseña el CSS para una tienda deportiva urbana estilo streetwear. Usa fuente Bebas Neue para títulos y DM Sans para el cuerpo. Paleta de colores: negro, blanco, gris claro y verde lima (#c8f135) como acento. Aplica Flexbox en header, grid de productos y formulario. Agrega hover effects que muestren botones deslizándose desde abajo en las tarjetas de producto."

---

### Prompt 3 — Ticker animado con CSS
**¿Para qué?** Agregar una banda de texto animado debajo del hero.

> "Crea una banda de texto que se desplace horizontalmente de forma infinita usando CSS puro con @keyframes. El texto debe repetirse para que el movimiento sea continuo y sin saltos. Fondo verde lima, texto negro, estilo tipográfico display en mayúsculas."

---

### Prompt 4 — Lookbook con CSS Grid
**¿Para qué?** Crear una galería de fotos con layout asimétrico.

> "Diseña una galería de lookbook con CSS Grid de 4 imágenes en layout asimétrico: una imagen alta que ocupa 2 filas, dos imágenes normales y una imagen ancha que ocupa 2 columnas. Todas con hover que hace zoom suave a la imagen."

---

### Prompt 5 — Formulario de contacto con foto de equipo
**¿Para qué?** Humanizar la sección de contacto con una imagen real del equipo.

> "Crea una sección de contacto que tenga a la izquierda información del equipo incluyendo una foto circular de perfil de Unsplash, nombre del equipo, horario de atención, dirección, teléfono y email. A la derecha un formulario con campos nombre, email, asunto y mensaje, con estilo limpio y minimalista."

---

### Prompt 6 — README completo del proyecto
**¿Para qué?** Documentar el proyecto para GitHub incluyendo todos los prompts.

> "Escribe el README.md para un proyecto de tienda deportiva streetwear llamada STRK. Incluye descripción, estructura de carpetas, tabla de secciones con etiquetas HTML, flujo de trabajo Git con 4 ramas, tecnologías usadas y lista detallada de 6 prompts utilizados durante el desarrollo indicando para qué se usó cada uno."
